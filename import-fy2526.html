Appraisal Tracker — One-Time FY2025-26 Import

Seeds the Roster and the Prior_Annual_Review tab from Master_PMS.xlsx. Run this once.

Before you run this

This pushes 164 employees into your live Appraisal Tracker spreadsheet. It is safe to re-run — Roster rows are matched and updated by ID (no duplicates), and Prior_Annual_Review rows are fully replaced for FY2025-26 each time you run it.

Manager and Email are intentionally blank for all 164 imported employees. Until HR/HODs fill those in (directly in the Roster sheet, or via a future form), these employees cannot log into Self-Appraisal or Manager Review — only the read-only FY25-26 snapshot will be available for them.

1. Connection

Paste your Apps Script Web App URL and access token (same ones used by your other tracker pages).

2. Import Roster ROSTER_COUNT employees

Creates 164 employee records with ID, Name, Designation, Department. Manager, Email left blank.

3. Import Prior Annual Review PAR_COUNT employees

Creates the FY2025-26 snapshot (CTC, KRA Professional, 360 Review, Total KRA, HOD proposal, promotion notes) for each employee, linked by the same Employee ID used in Roster. Run Step 2 first.
